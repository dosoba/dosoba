문제2> 풍선터치기

모든 행라인과 열라인에는 각각 1개 또는 0개의 풍선이 있습니다.
각 풍선들은 상,하,좌,우 방향으로 1초에 한번씩 이동할 수 있습니다.

다만, 아래와 같은 경우에는 방향이 변경됩니다.
첫번째, 상,하,좌,우 방향으로 이동하다가 벽에 부딪히는 경우에는 방향이 반대방향으로 변경됩니다.
예를들면 상으로 이동하다가 벽에 부딪히면 하방향으로 변경됩니다.
두번째, 풍선끼리 부딪히면 방향이 전환됩니다.
예를들면 상으로 이동하는 풍선과 우로 이동하는 풍선이 부딪히는 경우
상으로 이동하는 풍선으로 하방향으로 변경되고, 우로 이동하는 풍선의 경우 좌방향으로 변경됩니다.

일정한 시간(초)가 지난후에 사격 선수가 열 라인으로 이동해서 사격을 합니다.
이때 가장 많은 풍선을 터뜨리면 우승하는 게임입니다. 총알은 1게임당 1발만 사용할 수 있습니다.
다만, 선수는 100초가 지나기전에 꼭 1발은 발사를 해야 합니다.

100초 이내로 사격 선수가 최대한 많은 풍선을 터뜨릴수 있는 풍선의 개수를 구하는 프로그램을 작성하시오.

행라인과 열라인의 범위는 5 <= N <= 100 입니다.
최대 풍선의 개수는 1 <= K <= 2N 입니다.
입력에서 방향은 1 상, 2 하, 3 좌, 4 우 입니다.
사격 선수는 꼭 100초 이내에 반드시 사격을 해야 합니다.
50개 테스트케이스를 c언어기준 2초



[입력 샘플]
5			// 총 tc 수
5 5			// 행(열)의 수, 풍선의 개수
3 0 4		// y x 방향
1 1 2		// y x 방향
0 2 2		// y x 방향
2 3 3		// y x 방향
4 4 1		// y x 방향
