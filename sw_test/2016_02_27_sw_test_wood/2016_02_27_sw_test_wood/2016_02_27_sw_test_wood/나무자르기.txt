문제2> 나무자르기

아파트를 짓기 위해 필요한 나무를 제작해서 판매를 하고 있다.
산에서 벌목한 나무는 나무를 보관하는 박스에 보관을 하고 있다.
박스는 3개이상부터 10개까지 존재한다. (3<= N <= 10)
1개의 박스에는 최소 1개부터 20개까지 나무가 보관되어 있다. (1 <= box <= 20)
각 박스에는 일정 크기의 나무들이 존재한다. (1<= wood <= 1000)
각 박스에 있는 나무들은 그냥 판매를 할 수가 없다. 모두 동일한 크기로 가공을 해서
판매를 해야 한다. 

예를 들면 다음과 같다.
총 3개의 박스가 있다. 나무를 1번 자르는 비용은 1이고 단위 크기를 팔때 이윤은 17이다.
1번 박스 : 총 3개 6 12 18
2번 박스 : 총 1개 13
3번 박스 : 총 1개 15

3개의 박스에 들어있는 나무를 모두 일정한 크기(k)로 잘라서 판매를 할 경우 이윤이 최대가
되는 크기를 구해야 한다. 

단위크기를 3으로 할 경우,
1번 박스 나무토막 개수	: 2 / 4 / 6
1번 박스 나무자르는 횟수	: 1 / 3/ 5
2번 박스 나무토막 개수	: 4
2번 박스 나무자르는 횟수	: 4
3번 박스 나무토막 개수	: 5
3번 박스 나무자르는 횟수	: 4
단위나무 길이를 3으로 할 경우, 총 수익은 다음과 같다
총 수익	= (나무토막 개수 * 단위 길이 * 단위도막가격) - (자르는횟수 * 자르는 비용) 
		= 21 * 3 * 17 - (17 * 1)
		= 1071 - 17 = 1054
단위길이가 3일 때 최대수익은 1054이다.

단위크기를 6으로 할 경우,
1번 박스 나무토막 개수	: 1 / 2 / 3
1번 박스 나무자르는 횟수	: 0 / 1/ 2
2번 박스 나무토막 개수	: 2
2번 박스 나무자르는 횟수	: 2
3번 박스 나무토막 개수	: 2
3번 박스 나무자르는 횟수	: 2
단위나무 길이를 3으로 할 경우, 총 수익은 다음과 같다
총 수익	= (나무토막 개수 * 단위 길이 * 단위도막가격) - (자르는횟수 * 자르는 비용) 
		= 10 * 6 * 17 - (7 * 1)
		= 1013
단위길이가 6일 때 최대수익은 1013이다.

다만 나무를 자르지 않을 수도 있다.
자르는 비용과 수익이 다음과 같은 경우는 자르지 않아도 된다.
자르는 비용 57, 수익 1 이고,
다음과 같은 박스가 있다.
1번 박스 : 총 3개 6 12 18
2번 박스 : 총 1개 12
3번 박스 : 총 1개 12
위의 경우 자르는 비용이 크기 때문에 자르지 않는 경우가 최대 수익을 얻을 수 있다.

단위 크기를 12로 하고
1번 박스 나무토막 개수	: 0 / 1 / 0
1번 박수 나무자르는 횟수	: 0 / 0 / 0
2번 박스 나무토막 개수	: 1
2번 박스 나무자르는 횟수	: 0
3번 박스 나무토막 개수	: 1
3번 박스 나무자르는 횟수	: 0

총 수익  = 3 * 12 * 1 - 0 * 57 = 36
이경우는 36이 최대 수익이 될 수 있다. 

무조건 3개의 박스를 선택해서, 총 수익이 최대가 되는 단위길이를 선택해서 
최대 수익이 날 수 있도록 나무를 잘라서 공급하는 프로그램을 작성하시오.

[입력 샘플]
5			// 총 tc 수
5			// N개의 상자 (3<=N<=10)
1 17		// 자르는비용(c), 단위나무도막 수익(v)
1 25		// 나무도막의 갯수, 나무도막의 길이
1 51		// 나무도막의 개수, 나무도막의 길이
3 25 27 30	// 나무도막의 개수, 나무도막의 길이1, 나무도막의 길이2, 나무도막의 길이3
2 16 32		// 나무도막의 개수, 나무도막의 길이1, 나무도막의 길이2
1 22		// 나무도막의 개수, 나무도막의 길이1
