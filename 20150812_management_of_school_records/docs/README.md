# 성적 프로그램 1차
## 문제 정의

한 학년 학생들의 인적정보와 성적 정보를 관리하는 프로그램을 작성한다.

## 요구사항

학생들의 인적정보를 추가, 삭제, 수정, 조회 할 수 있어야 한다.
학생들의 과목정보 점수를 기재, 수정 할 수 있어야 한다.
과목을 추가, 수정, 삭제 조회 할 수 있어야 한다.
학생들의 성적 총합과 평균, 석차를 조회 할 수 있어야 한다.
